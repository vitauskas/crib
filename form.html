текстовые поля

Текст, любой тип символов

Email, может показывать предупреждение об ошибке

Пароль, символы отображаются как точки

Число, используются клавиши вверх вниз

Телефон, может сработать автозаполнение

Многострочный текст, может быть изменен размер поля.

placeholder

 подсказывающий текст, который изчезает при вводе

label

 текстовая метка к элементу формы

label (текстовую метку), можно связать с определенным
элементом формы с помощью атрибута for, соответствующему
значению id у поля. Таким образом, кликая на "имя" - попадаем
в окно элемента формы.

Но есть и более простой способ. Нужно поместить текст и
элемент формы внутрь метки label.

При этом абсолютно не важна форма размещения.

флажки

флажки имеют два состояния:
- включено/выключено
- да/нет

запомнить меня

здесь, тоже, если поместить всю конструкцию
внутрь label, то можно кликать и по тексту.

по умолчанию флажок выключен. Можно
пометить его включенным, используя атрибут
checked

преключатели

Предоставляют пользователю на выбор список вариантов.
Для этого нужно сгрупировать список при помощи
одинакового заначения атрибута name.




Эти переключатели являются взаимоисключающими.
Так как name одинаковый для всех переключателей -
выбор одного из вариантов отменит все остальные.

разница между переключателями и флажками

  1. флажок существует сам по себе
  2. переключатель существует ввиде списка (>2)
  3. выбор флажка произвольный
  4. выбор переключателя обязательный

выпадающее меню

если количество вариантов для выбора занимает
слишком много места, можно воспользоваться выпадающим
меню select.

множественный выбор из выпадающего меня
- атрибут multiple, дает возможность выбора нескольких
вариантов.
